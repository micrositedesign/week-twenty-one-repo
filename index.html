What is a Domain Name?

Domain names are easy to remember strings of text that are mapped to an IP address in order to access a website found on a web server. For instance, one of the IPv4 IP addresses used by Google is 64.233.160.0, but instead of needing to remember this number, someone can just type in google.com and thanks to a domain name server (DNS), the website can be accessed more easily. People tend to remember words easier than numbers, so a domain name is often the preferred method for accessing websites.
